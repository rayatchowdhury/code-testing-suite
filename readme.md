## 🚀 Quick Start

1. **Clone the repository**:

   ```bash
   git clone https://github.com/yourusername/code-testing-suite.git
   cd code-testing-suite
   ```

2. **Install dependencies**:

   ```bash
   pip install -r requirements.txt
   ```

3. **Launch the application**:
   ```bash
   python main.py
   ```
